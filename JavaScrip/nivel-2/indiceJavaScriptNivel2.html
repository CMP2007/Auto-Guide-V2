Notas:

Para comentar líneas de código en JavaScript se añaden dos barras oblicuas (//) al principio de la línea, o también se puede agilizar el proceso seleccionando el código por comentar y cliqueando Control (Ctrl) + la llave de cierre ( } ).

A su vez existe una segunda forma de comentar el código en JavaScript, la cual es utilizando los símbolos "/*" y "*/" para delimitar el texto a comentar.
